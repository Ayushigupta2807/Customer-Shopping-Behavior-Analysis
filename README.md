
# Customer Shopping Behavior Analysis

##  Overview

End-to-end Data Analytics project analyzing customer shopping trends using retail data. This project uses SQL, Python, and Power BI to extract business insights and build an interactive dashboard.

##  Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* SQL
* Power BI

##  Key Highlights

* Performed data cleaning and exploratory data analysis
* Analyzed customer purchasing patterns
* Identified revenue-driving categories
* Built an interactive Power BI dashboard

##  Outcome

Demonstrates skills in data analysis, SQL querying, visualization, and business insight generation.
